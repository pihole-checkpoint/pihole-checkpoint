Adding Instances

Instances are auto-discovered from environment variables on container startup. To add a new instance, set the following env vars and restart:

PIHOLE_{PREFIX}_URL=https://...
PIHOLE_{PREFIX}_PASSWORD=...
PIHOLE_{PREFIX}_VERIFY_SSL=false

Replace {PREFIX} with any name (e.g., GYM, OFFICE).

Optional Settings

Override defaults via env vars:

PIHOLE_{PREFIX}_NAME=My Pi-hole
PIHOLE_{PREFIX}_SCHEDULE=daily
PIHOLE_{PREFIX}_TIME=03:00
PIHOLE_{PREFIX}_DAY=0
PIHOLE_{PREFIX}_MAX_BACKUPS=10
PIHOLE_{PREFIX}_MAX_AGE_DAYS=30
Removing Instances

Remove the PIHOLE_{PREFIX}_URL env var and restart. The instance will be marked as removed and scheduled backups will stop. Existing backups are retained as orphaned and can be downloaded or deleted individually.
