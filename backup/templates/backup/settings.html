Help
Environment Variables

Pi-hole connection settings are configured via environment variables in your .env file or Docker Compose.

Each instance uses a prefix to determine which env vars to read:

  • PIHOLE_{PREFIX}_URL - Pi-hole URL
  • PIHOLE_{PREFIX}_PASSWORD - Admin password
  • PIHOLE_{PREFIX}_VERIFY_SSL - SSL verification (default: false)

Example: prefix PRIMARY reads PIHOLE_PRIMARY_URL

Backup Frequency
  • Hourly: Every hour
  • Daily: Once per day at specified time
  • Weekly: Once per week on specified day/time
Retention Policy

Old backups are automatically deleted based on count and age limits. Set to 0 to disable a limit.
